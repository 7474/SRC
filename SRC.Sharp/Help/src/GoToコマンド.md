**GoToコマンド**

ラベルにジャンプ

**書式**

**GoTo** *label*

**指定項目説明**

*label*ジャンプ先のラベル名。「:」は不要

**解説**

ラベル*label* にジャンプしてコマンドの実行の流れを変更します。

![](./images/bm0.gif) *label* にイベントラベルを指定するときはラベルを必ず「"」で囲ってください。

**GoTo**コマンドで**For**ループやサブルーチン内にジャンプして入ったり(出たり)するとループの動作がおかしくなります。このようなジャンプはしないようにしてください。

**例**
```sh
全滅 敵:
GoTo 増援出現
```
