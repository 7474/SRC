収納イベント

ユニットが母艦に着艦したときに発生するイベント

書式

収納 unit :

指定項目説明

unit着艦したユニットのメインパイロット名、ユニット名、または陣営名

解説

ユニットunit が母艦に着艦した場合に発生するイベントです。Landコマンドでユニットを母艦に格納させた場合は発生しません。

収納先の母艦についての情報を得たい場合はシステム変数相手パイロットまたは相手ユニットＩＤを使ってください。

例

#味方ユニットが母艦に着艦した場合

収納 味方:

#マップ上に出撃しているユニット数をカウント

出撃数 = 0

ForEach 味方 出撃

Incr 出撃数

Next

#出撃数が1ということはこのユニットが最後のユニットなのでステージを終了

If 出撃数 = 1 Then

Talk ロイド艦長

よし、全員着艦したな。戦線を離脱する！

End

#次のステージに移行

Continue Next.eve

EndIf

#イベントを終了

Exit
